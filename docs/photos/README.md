# Photos

Drop project photos here — the main README references these files:

| Filename | What it shows |
|---|---|
| `01_atom_matrix_on.jpg` | Atom Matrix showing solid green (compressor ON) mounted on a metal cabinet |
| `02_compressor_lever.jpg` | Servo mechanism on the compressor's OFF/ON lever |
| `03_workshop_overview.jpg` | Wide shot of the workshop — compressor, StamPLC on wall, cooling fan, dryer |
| `04_stamplc_ssr_close.jpg` | StamPLC on DIN rail flanked by two Fotek SSR-40 DA-H — LCD shows green ON |
| `05_stamplc_ssr_clean.jpg` | Clean close-up of the same, well-lit — used as the top hero image |

## How to add them

1. Save your photos with the filenames above (or update the paths in
   [`../../README.md`](../../README.md)).
2. Aim for **1600 px on the long side** to keep the repo lightweight.
3. Commit and push:
   ```bash
   git add docs/photos/*.jpg
   git commit -m "Add project photos"
   git push
   ```

Feel free to add more shots — installation progress, close-ups of wiring, etc.
GitHub renders JPEG and PNG inline.
